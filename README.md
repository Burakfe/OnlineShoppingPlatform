# Kefeshop - Frontend Demo Prototype

This is a simple frontend-only demo for the CMPE 313 / SENG 214 Kefeshop / Online Shopping Platform project.

## What it demonstrates

- Customer login/register
- Product browsing
- Search and category filtering
- Product detail page
- Add/remove/update cart
- Checkout simulation
- Payment success/failure simulation
- Order confirmation
- Order tracking
- Review submission
- Admin dashboard
- Product management
- Order status management
- User management
- Role-based access control simulation
